\doxysubsection{Detailed Description}
The \mbox{\hyperlink{classWeatherRecord}{Weather\+Record}} class holds weather data for one city. 

The \mbox{\hyperlink{classWeatherRecord}{Weather\+Record}} class holds weather data in the form of \mbox{\hyperlink{classDailyWeather}{Daily\+Weather}} for one city. It is also an interface for classes like \mbox{\hyperlink{classWeatherWindow}{Weather\+Window}} and \mbox{\hyperlink{classWeatherCategory}{Weather\+Category}} to interact with \mbox{\hyperlink{classDailyWeather}{Daily\+Weather}} data without needed to know the underlying implementaation of \mbox{\hyperlink{classDailyWeather}{Daily\+Weather}} \begin{DoxyAuthor}{Author}
Adam Miller 
\end{DoxyAuthor}
