\doxysubsection{Detailed Description}
The \mbox{\hyperlink{classWeatherRecord}{Weather\+Record}} class holds weather data for one city. 

The \mbox{\hyperlink{classWeatherRecord}{Weather\+Record}} class holds weather data in the form of \mbox{\hyperlink{classDailyWeather}{Daily\+Weather}} for one city.
