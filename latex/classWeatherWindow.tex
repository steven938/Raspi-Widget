\doxysubsection{Detailed Description}
The \mbox{\hyperlink{classWeatherWindow}{Weather\+Window}} class displays weather data. 

The \mbox{\hyperlink{classWeatherWindow}{Weather\+Window}} class interacts with \mbox{\hyperlink{classWeatherCategory}{Weather\+Category}} to present weather data. It takes in input from the user and presents output back to them.
