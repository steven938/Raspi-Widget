\doxysubsection{Detailed Description}
The \mbox{\hyperlink{classMainWindow}{Main\+Window}} class is the primary window of the application. 

The \mbox{\hyperlink{classMainWindow}{Main\+Window}} is the priamry window the application, and provides the user the ability to access Stock\+Window, \mbox{\hyperlink{classWeatherWindow}{Weather\+Window}}, and \mbox{\hyperlink{classNewsWindow}{News\+Window}}
